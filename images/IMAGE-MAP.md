# Where each photo goes — save your photos into THIS folder

The site is already wired to these exact filenames. Save each photo you sent me
into this `images/` folder using the matching name below, and it will appear
automatically. Until a file exists, that spot just shows a solid green block, so
nothing ever looks broken.

JPG is assumed (`.jpg`). If you save as PNG, keep the name but the site expects
`.jpg`, so either export as JPG or tell me and I'll switch the references.

  handles any shape.
